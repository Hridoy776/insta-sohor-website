Difference between local storage and session storage

local storage

locat storage is used for storage data locally in user's browser.This data remains in browser if it is cleaned by setting or manually by users. local storage data will remain even through page relode or going to another tab

session storage

session storage also work like local storage to storage data but it's expiration time is end with the page session close.This data will clear when users go on another tab or another page.
